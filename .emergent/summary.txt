<analysis>**original_problem_statement**: Build a secure, therapist-first web application for managing a therapy practice and supporting clinical decision-making.

**PRODUCT REQUIREMENTS**
- **User Roles**: Therapist, Client, Super Admin, and Therapist Assistant.
- **Authentication**: Secure JWT-based login for all roles.
- **Practice Management**: Client profiles, appointment scheduling, session notes (SOAP & DAP), secure messaging, and payment tracking with printable receipts.
- **Clinical Support**: AI-powered assessment suggestion engine, therapy protocol builder, and homework/resource sharing.
- **Super Admin & Subscriptions**: A Super Admin role to manage therapists, clients, and a subscription module.
- **Therapist Assistant Role**: A restricted role to manage operational tasks (like session check-in/out and payments) without access to clinical data.
- **Clinical Workflow**: A mandatory **Case History** must be completed by the therapist before any session notes can be created, followed by a client-signed **Therapy Consent** form.
- **Client Self-Service**: Clients should be able to view their dashboard, book appointments, and view payment history.
- **UI/UX**: Profile photo support, search functionality, a comprehensive client profile view, and a customizable theme picker for all users.
- **Global Standards**: The application must use India Standard Time (IST), DD/MM/YYYY date format, 24-hour (HH:mm) time format, and Indian Rupee (₹) currency.

**User's preferred language**: English

**what currently exists?**
A full-stack application with a FastAPI backend, React frontend, and MongoDB. Key features include:
-   **Multi-Role Authentication**: Secure login for Super Admin, Therapist, Client, and Therapist Assistant.
-   **Complete Clinical Workflow**: A mandatory Case History wizard for therapists, automated consent form generation, and a client-side digital signing flow.
-   **AI-Powered Clinical Suite**: Integrated with Gemini 3 Flash for assessment suggestions, protocol building, and resource sharing.
-   **Comprehensive Client Profile**: A modal view for therapists that aggregates all client data, including appointments, session notes, and case history, with integrated actions like booking appointments and starting new notes.
-   **Session & Payment Management**: A full check-in/check-out workflow for appointments, which triggers a payment capture dialog. The system generates downloadable/printable HTML receipts for payments.
-   **Functional Client Dashboard**: Clients can view their upcoming appointments, book new appointments, and see their payment history.
-   **UI Customization**: A theme picker in the settings allows users (Therapist, Client, Admin) to choose from 5 predefined color themes, with the choice persisted in the database.
